桃太郎

ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
1. 起
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー

むかしむかし、あるところにおじいさんとおばあさんが住んでいました。
おじいさんは山に柴刈りに、おばあさんは川に洗濯に行きました。

おばあさんが洗濯していると、川上からどんぶらこどんぶらこと、大きな桃が流れてきました。
桃をもって帰り、山から帰ったおじいさんと一緒に食べようとすると、
桃が二つに割れ中から元気そうな男の子が出てきました。
子どものいないおじいさんとおばあさんは大喜びで、桃太郎と名前を付けとても大事に育てました。


ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
2. 承
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー

大きくなった桃太郎は、鬼ヶ島に鬼の征伐に出かけることになり、おばあさんがきび団子を作ってくれました。
鬼ヶ島に向かう途中、ドーベルマンが出てきて

「一つくださいきび団子。お供についていきましょう。」

次にゴリラが来て

 「一つくださいきび団子。お供についていきましょう。」

今度は鷹が飛んできて

「一つくださいきび団子。お供についていきましょう。」

ドーベルマン、ゴリラ、鷹の３匹は桃太郎の家来になり、船に乗って鬼ヶ島へ向かいました。


ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
3. 結 
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー


鬼ヶ島で酒盛りをしていた鬼たちに、ドーベルマンはかみつき、ゴリラはひっかき、鷹はつっつきました。
桃太郎は金棒を持った鬼の大将をやっつけました。

「もう悪いことはしないか。」「もうしません。」

鬼が悪いことをして集めた宝物を車に積み、
ドーベルマンが車を引き、鷹が綱を引っ張り、ゴリラが後押しをしてもち帰りました。
