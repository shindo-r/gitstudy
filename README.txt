桃太郎

ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
1. 起
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー

むかしむかし、あるところにおじいさんとおばあさんが住んでいました。
おじいさんは山に柴刈りに、おばあさんは川に洗濯に行きました。

おばあさんが洗濯していると、川上からどんぶらこどんぶらこと、大きな桃が流れてきました。
桃をもって帰り、山から帰ったおじいさんと一緒に食べようとすると、
桃が二つに割れ中から元気そうな男の子が出てきました。
子どものいないおじいさんとおばあさんは大喜びで、桃太郎と名前を付けとても大事に育てました。


ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
2. 承
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー

大きくなった桃太郎は、鬼ヶ島に鬼の征伐に出かけることになり、おばあさんが現金を用意してくれました。
鬼ヶ島に向かう途中、犬が出てきて

「一つください現金を。お供についていきましょう。」

次に猿が来て

「一つください現金を。お供についていきましょう。」

今度はキジが飛んできて

「一つください現金を。お供についていきましょう。」

犬、猿、キジの３匹は桃太郎の家来になり、船に乗って鬼ヶ島へ向かいました。


ーーーーーーーーーーーーーーーーーーーーーーーーーーーー
3. 結 
ーーーーーーーーーーーーーーーーーーーーーーーーーーーー


鬼ヶ島で酒盛りをしていた鬼たちに、犬はかみつき、猿はひっかき、キジはつっつきました。
桃太郎は金棒を持った鬼の大将をやっつけました。

「もう悪いことはしないか。」「もうしません。」

鬼が悪いことをして集めた宝物を車に積み、
犬が車を引き、キジが綱を引っ張り、猿が後押しをしてもち帰りました。
